Instalasi Debian

Sebelum menginstall Debian, pastikan kalian sudah memiliki file ISO dari Debian itu, jika belum, kalian bisa mengunduhnya di sini


Apabila kalian sudah memiliki file ISO, kalian bisa melanjutkan ke Langkah-langkah berikut

1. Buka VirtualBox

2. Pada navigasi pilih opsi “Baru”

3. Beri nama pada sistem operasi kita

4. Pada kolom folder, klik panah kebawah dan pilih opsi lainnya

5. Setelahnya kita akan masuk ke File Manager, kemudian buatlah folder baru sebagai tempat penyimpanan sistem operasi kita. Kemudian pilih folder tersebut

6. Setelahnya ke kolom image ISO, klik pilih ISO dan pilih ISO debian kita

7. Setelahnya, centang pada bagian instalasi tak terdampingi

8. Pilih Hard Disk dan saya menggunakan pengaturan ini

9. Kemudian klik selesai dan pilih opsi mulai di navigasi

10. Tampilan awal kurang lebih akan seperti ini

11. Untuk tampilan yang lebih besar, pilih opsi Tilik > Layar Visual > dan saya memilih layar virtual 150%

12. Setelahnya tampilannya akan seperti ini

13. Karena mesin virtual ini tidak bisa memakai krusor, kita perlu menggunakan tombol PgUp, PgDn, Home, dan End. Untuk opsi ini pilih Install

14. Kemudian pilih English untuk opsi ini

15. Kemudian untuk Location pilih United State saja

16. Untuk keymap gunakan American English

17. Setelahnya masukkan nama, ketika sudah dirasa cocok, klik Continue

18. Kemudian untuk bagian Domain Name kita kosongkan saja

19. Kemudian ini adalah bagian yang cukup krusial, yaitu kata sandi untuk Root, disini akan saya samakan saja dengan kata sandi biasa, tujuannya agar lebih mudah diakses

20. Kemudian masukkan kata sandi yang barusan kita buat

21. Kemudian kita akan memasukkan nama pengguna

22. Kemudian kita akan memasukkan Username, disini saya menggunakan sesuai apa yang diberikan oleh sistem

23. Kemudian kita akan membuat password untuk masuk ke mode Super User, disini saya menggunakan password yang sama seperti sebelumnya

24. Setelahnya ulangi password untuk mengkonfirmasi

25. Kemudian untuk Time Zone pilih opsi Eastern

26. Opsi partisi pilih bagian ini

27. Untuk bagian Disk Partition pilih opsi ini

28. Untuk Partition Schema pilih opsi ini

29. Setelahnya pilih opsi ini untuk memulai penulisan ulang partisi

30. Setelahnya pilih ini untuk mengubah partisinya

31. Untuk opsi ini saya memilih untuk menolaknya

32. Setelahnya kita akan diahapkan dengan pilihan Software Selection seperti ini

33. Saya memakai preferensi seperti ini

34. Setelahnya kita akan menunggu proses instalasi yang Boot Loader

35. Saya memilih opsi ini untuk Boot Loader saya

36. Kemudian pilih opsi ini

37. Kemudian tunggu, biasanya proses ini memakan banyak waktu

38. Setelahnya pilih Continue untuk melakukan proses Reboot

39. Kemudian layar kita akan berubah menjadi hitam, hal ini normal karena kita akan mulai masuk ke Debian CLI (Command Line Interfaces)

40. Kemudian masuk menggunakan akun yang sudah kita buat
